ソースコードからインストールする方法です。

OS は Linux や macOS を想定しています。

git clone git@github.com:git/git.git
cd git
make configure
./configure --prefix=/usr/local/git
sudo make install
git --version

ライブラリ依存エラーが発生する場合は、別途調べてみてください。

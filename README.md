# Osync I/O

Hugo content for [osync.io](https://osync.io).

## Installing Hugo

[Download Hugo](https://github.com/gohugoio/hugo/releases).

## Development

Running the local development server.

```sh
hugo --cleanDestinationDir server --bind 0.0.0.0
```

## Build static pages

```sh
hugo --cleanDestinationDir
```
